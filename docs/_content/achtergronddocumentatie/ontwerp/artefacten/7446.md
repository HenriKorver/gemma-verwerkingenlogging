### Functionaliteit

De functie maakt een verwerkingsactie in het verwerkingenlog.
De verwerkingsactie is **niet** vertrouwelijk.

### API call

POST /verwerkingsacties

Aanroep:
* Autorisatiescope = `create:normal`
* Parameter Vertrouwelijk = 'Normaal'
