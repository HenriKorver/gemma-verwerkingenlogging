### Functionaliteit

De functie maakt een verwerkingsactie in het verwerkingenlog en markeert deze als vertrouwelijk.

### API call

POST /verwerkingsacties

Aanroep:
* Autorisatiescope = `create:confidential`
* Parameter Vertrouwelijkheid = 'Vertrouwelijk'
